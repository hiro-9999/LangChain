1. Mac標準の音声入力を有効にする手順
まずは設定を確認しましょう。

**「システム設定（System Settings）」**を開きます。

左側のメニューから**「キーボード（Keyboard）」**を選択します。

右側の画面を下にスクロールして、**「音声入力（Dictation）」**セクションを探します。

「音声入力」をオンにします。

**「ショートカット（Shortcut）」**を確認します（デフォルトでは 「マイクボタン 🎙️」 か 「Fnキーを2回押す」 になっていることが多いです）。

2. .txt ファイルで入力する実践ステップ
テキストエディット（TextEdit） または任意のテキストエディタで .txt ファイルを開きます。

文字を入力したい場所にカーソルを置きます。

先ほど確認した ショートカット（例：Fnキーを2回押す） を実行します。

画面にマイクのアイコンが表示されたら、話し始めます。

話し終わったら、もう一度ショートカットを押すか、完了ボタンを押します。

💡 便利なコツ（日本語・中国語共通）
句読点も声で入力: 「まる（。）」や「てん（、）」、「改行（かいぎょう）」と言うだけで入力してくれます（中国語なら "句号" "逗号" "换行"）。

言語の切り替え: 「音声入力」の設定画面で「言語（Languages）」を追加しておけば、日本語と中国語（あるいは英語）を混ぜて入力することも可能です（Deliver Results）。
