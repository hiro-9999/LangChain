“CSV データビジュアライザー”という名前のミニアプリReactアーティファクトを作成したいと思います。機能としては、ユーザーが.csvファイルをアップロードします。アプリはデータの最初の5行を表示します。最初の行は列ヘッダーと仮定します。その後、ユーザーが実行したい分析を記述するためのテキストボックスがあります。分析の4つの出力形式は、棒グラフ、折れ線グラフ、テーブル、またはカウントです。

実際に分析を実行するには、Claude APIを呼び出して、要求された分析を行うJavaScriptを生成します。Claude APIがJavaScriptを返したら、ブラウザでそれを実行して分析を実装します。Claude APIからの出力トークン数は16kに設定してください。Claude APIからの応答を待っている間や計算を実行している間は、スピナーを表示します。APIを呼び出している間はスピナーの下に“分析を計画中”というラベルを表示し、JavaScriptを実行している間は“結果を計算中”にラベルを変更します。計算が完了したらスピナーを非表示にします。Claudeへのプロンプトには、データの最初の5行のサンプルを含める必要があります。

analysisResult = { 
  type: ‘bar_chart’ | ‘line_chart’ | ‘table’ | ‘count’, 
  data: <適切なデータ構造>, 
  config: <オプションの構成オブジェクト> 
};

【データ構造（data）の定義】

1. 棒グラフ (bar_chart)
棒グラフを描画するための形式です。
* labels (string[]): 横軸に表示する項目の名前。文字列の配列で指定します。
* values (number[]): 各項目に対応する数値の配列。
* title (string, 任意): グラフのタイトル。
例: { labels: ["1月", "2月"], values: [100, 200], title: "月別売上" }

2. 折れ線グラフ (line_chart)
時系列などの推移を表すための形式です。構造は棒グラフと同じです。
* labels (string[]): X軸（横軸）のラベル。
* values (number[]): Y軸（縦軸）のプロット値。
* title (string, 任意): グラフのタイトル。

3. 表 (table)
データをグリッド状に整理して表示するための形式です。
* headers (string[]): 表の一番上の見出し（列名）の配列。
* rows (any[][]): 各行のデータの配列。配列の中に配列が入った「二次元配列」の形式になりま
す。
例: headers: ["名前", "年齢"] rows: [ ["田中", 25], ["佐藤", 30] ]

4. カウント (count)
合計値や達成率など、特定の数字を強調して表示するための形式です。
* value (number): 表示したいメインの数値。
* label (string, 任意): その数値が何を表しているかのラベル（例：「合計ユーザー数」など）。

プロンプトには以下の追加要件も含める必要があります。

* 分析に基づいてもっとも適切な可視化タイプを選択する
* 数値集計（合計、平均、カウント）には‘count’タイプを使用する
* カテゴリー間の比較には‘bar_chart’を使用する
* 時系列のトレンドには‘line_chart’を使用する
* 詳細なデータ検査やピボットテーブルには‘table’を使用する
* 適切な場所に意味のあるラベルとタイトルを含める
* エッジケース（空のデータ、欠損値など）を適切に処理する

アプリのデザインは白い背景に青いボタンを使用し、プロフェッショナルにデザインされたデータ分析ツールのような見た目にする必要があります。

import React, { useState } from 'react';
import { Plus, Trash2, CheckCircle, Circle, ListChecks } from 'lucide-react';

const TodoApp = () => {
  // ① 状態管理：TODOリスト本体
  const [todos, setTodos] = useState([
    { id: 1, text: 'サンプルタスク', completed: false }
  ]);
  
  // ② 状態管理：入力中のテキスト
  const [inputText, setInputText] = useState('');

  // ③ 関数：タスクの追加
  const addTodo = () => {
    if (inputText.trim() === '') return;
    const newTodo = {
      id: Date.now(),
      text: inputText,
      completed: false
    };
    setTodos([...todos, newTodo]);
    setInputText('');
  };

  // ④ 関数：完了状態の切り替え
  const toggleTodo = (id) => {
    setTodos(todos.map(todo => 
      todo.id === id ? { ...todo, completed: !todo.completed } : todo
    ));
  };

  // ⑤ 関数：タスクの削除
  const deleteTodo = (id) => {
    setTodos(todos.filter(todo => todo.id !== id));
  };

  // ⑥ 統計データの算出（前述のcount構造に準拠）
  const stats = {
    total: { value: todos.length, label: "合計タスク" },
    completed: { value: todos.filter(t => t.completed).length, label: "完了済み" }
  };

  return (
    <div className="max-w-md mx-auto my-10 p-6 bg-white rounded-2xl shadow-xl font-sans">
      {/* ヘッダーと統計表示 */}
      <div className="flex items-center gap-2 mb-6 border-b pb-4">
        <ListChecks className="text-blue-500" />
        <h1 className="text-2xl font-bold text-gray-800">My Tasks</h1>
      </div>

      <div className="grid grid-cols-2 gap-4 mb-8">
        <div className="bg-blue-50 p-4 rounded-xl text-center">
          <p className="text-2xl font-bold text-blue-600">{stats.total.value}</p>
          <p className="text-xs text-blue-400">{stats.total.label}</p>
        </div>
        <div className="bg-green-50 p-4 rounded-xl text-center">
          <p className="text-2xl font-bold text-green-600">{stats.completed.value}</p>
          <p className="text-xs text-green-400">{stats.completed.label}</p>
        </div>
      </div>

      {/* 入力エリア */}
      <div className="flex gap-2 mb-6">
        <input
          type="text"
          className="flex-1 p-3 border rounded-lg focus:outline-none focus:ring-2 focus:ring-blue-400"
          placeholder="新しいタスクを入力..."
          value={inputText}
          onChange={(e) => setInputText(e.target.value)}
          onKeyPress={(e) => e.key === 'Enter' && addTodo()}
        />
        <button onClick={addTodo} className="p-3 bg-blue-500 text-white rounded-lg hover:bg-blue-600 transition-colors">
          <Plus size={24} />
        </button>
      </div>

      {/* リスト表示エリア */}
      <div className="space-y-3">
        {todos.map(todo => (
          <div key={todo.id} className="flex items-center justify-between p-3 border rounded-lg hover:bg-gray-50 transition-colors group">
            <div className="flex items-center gap-3 cursor-pointer" onClick={() => toggleTodo(todo.id)}>
              {todo.completed ? <CheckCircle className="text-green-500" /> : <Circle className="text-gray-300" />}
              <span className={`${todo.completed ? 'line-through text-gray-400' : 'text-gray-700'}`}>
                {todo.text}
              </span>
            </div>
            <button onClick={() => deleteTodo(todo.id)} className="text-gray-300 hover:text-red-500 transition-colors">
              <Trash2 size={18} />
            </button>
          </div>
        ))}
      </div>
    </div>
  );
};

export default TodoApp;

          // マークダウンのコードブロックを除去
          htmlContent = htmlContent.replace(/```html\n?/g, '').replace(/```\n?/g, '');
          
          // ファイルを更新
          setFiles(files.map(f => 
            f.id === fileId ? { 
              ...f, 
              status: '変換完了', 
              prototypeHtml: htmlContent,
              error: null
            } : f
          ));
          
          // 選択中のファイルも更新
          if (selectedFile?.id === fileId) {
            setSelectedFile({
              ...selectedFile,
              status: '変換完了',
              prototypeHtml: htmlContent,
              error: null
            });
          }
        } else {
          throw new Error('APIレスポンスが不正です');
        }
      } catch (error) {
        console.error('変換エラー:', error);
        setFiles(files.map(f => 
          f.id === fileId ? { 
            ...f, 
            status: 'エラー', 
            error: error.message || 'プロトタイプの生成に失敗しました'
          } : f
        ));
        
        if (selectedFile?.id === fileId) {
          setSelectedFile({
            ...selectedFile,
            status: 'エラー',
            error: error.message || 'プロトタイプの生成に失敗しました'
          });
        }
      } finally {
        setIsProcessing(false);
        setProcessingId(null);
      }
    };
    
    reader.readAsDataURL(file.file);
  };

  const deleteFile = (fileId) => {
    setFiles(files.filter(f => f.id !== fileId));
    if (selectedFile?.id === fileId) {
      setSelectedFile(null);
    }
  };

  const downloadPrototype = (file) => {
    if (!file.prototypeHtml) return;
    
    const element = document.createElement('a');
    const blob = new Blob([file.prototypeHtml], { type: 'text/html' });
    const url = URL.createObjectURL(blob);
    element.href = url;
    element.download = file.name.replace('.pdf', '') + '_prototype.html';
    document.body.appendChild(element);
    element.click();
    document.body.removeChild(element);
    URL.revokeObjectURL(url);
  };

  const totalFiles = files.length;
  const convertedFiles = files.filter(f => f.status === '変換完了').length;

  return (
    <div className="p-4 max-w-7xl mx-auto bg-gradient-to-br from-purple-50 to-blue-50 min-h-screen font-sans">
      <header className="mb-8 bg-gradient-to-r from-purple-600 to-blue-600 p-8 rounded-2xl text-white shadow-2xl">
        <div className="flex items-center justify-between">
          <div>
            <h1 className="text-3xl font-bold flex items-center gap-3 mb-2">
              <Sparkles size={32} /> PDF2Prototype
            </h1>
            <p className="text-purple-100 text-sm">AIがPDFデザインを解析し、動作するプロトタイプを自動生成</p>
          </div>
          <div className="text-right bg-white/20 backdrop-blur-sm px-6 py-4 rounded-xl">
            <p className="text-xs opacity-90 mb-1">変換済みファイル</p>
            <p className="text-3xl font-bold">{convertedFiles}/{totalFiles}</p>
          </div>
        </div>
      </header>

      <div className="grid grid-cols-5 gap-6">
        {/* 左側：アップロードとファイルリスト */}
        <div className="col-span-2 space-y-6">
          <section className="bg-white p-6 rounded-2xl shadow-lg border-2 border-dashed border-purple-200 hover:border-purple-400 transition-all">
            <h2 className="text-lg font-semibold mb-4 flex items-center gap-2 text-gray-800">
              <Upload size={22} className="text-purple-600" /> PDFファイルをアップロード
            </h2>
            <label className="cursor-pointer block">
              <input 
                type="file" 
                accept=".pdf"
                multiple
                className="hidden"
                onChange={handleFileUpload}
              />
              <div className="bg-gradient-to-br from-purple-50 to-blue-50 p-8 rounded-xl text-center border-2 border-purple-300 border-dashed hover:bg-purple-100 transition-all">
                <Upload size={40} className="mx-auto mb-3 text-purple-500" />
                <p className="font-bold text-gray-700 mb-1 text-sm">クリックまたはドラッグ&ドロップ</p>
                <p className="text-xs text-gray-500">PDFファイルをアップロード（複数可）</p>
              </div>
            </label>
          </section>

          <section className="space-y-3">
            <h2 className="text-lg font-semibold text-gray-700 px-2 flex items-center gap-2">
              <FileText size={20} /> アップロード済みファイル
            </h2>
            {files.length === 0 ? (
              <div className="bg-white p-6 rounded-xl text-center text-gray-400">
                <FileText size={40} className="mx-auto mb-2 opacity-30" />
                <p className="text-sm">まだファイルがアップロードされていません</p>
              </div>
            ) : (
              files.map(file => (
                <div 
                  key={file.id} 
                  className={`bg-white p-4 rounded-xl shadow-md border-2 transition-all cursor-pointer ${
                    selectedFile?.id === file.id 
                      ? 'border-purple-500 shadow-lg' 
                      : 'border-gray-100 hover:border-purple-300'
                  }`}
                  onClick={() => setSelectedFile(file)}
                >
                  <div className="flex items-center justify-between mb-3">
                    <div className="flex items-center gap-3 flex-1 min-w-0">
                      <div className={`p-2 rounded-lg flex-shrink-0 ${
                        file.status === '変換完了' 
                          ? 'bg-green-100 text-green-600' 
                          : file.status === 'エラー'
                          ? 'bg-red-100 text-red-600'
                          : 'bg-purple-100 text-purple-600'
                      }`}>
                        <FileText size={20} />
                      </div>
                      <div className="flex-1 min-w-0">
                        <p className="font-bold text-gray-800 text-sm truncate">{file.name}</p>
                        <p className="text-xs text-gray-500">{file.size} KB</p>
                      </div>
                    </div>
                  </div>
                  
                  <div className="flex items-center gap-2">
                    <span className={`px-2 py-1 rounded-full text-xs font-semibold flex-shrink-0 ${
                      file.status === '変換完了' 
                        ? 'bg-green-100 text-green-700' 
                        : file.status === 'エラー'
                        ? 'bg-red-100 text-red-700'
                        : 'bg-yellow-100 text-yellow-700'
                    }`}>
                      {file.status}
                    </span>
                    
                    {file.status === '未処理' && (
                      <button
                        onClick={(e) => {
                          e.stopPropagation();
                          convertToPrototype(file.id);
                        }}
                        disabled={isProcessing}

                      >
                        <Wand2 size={14} /> 
                        {processingId === file.id ? '処理中...' : '変換'}
                      </button>
                    )}
                    
                    {file.status === '変換完了' && (
                      <button
                        onClick={(e) => {
                          e.stopPropagation();
                          downloadPrototype(file);
                        }}
                        className="bg-blue-600 text-white px-3 py-1 rounded-lg text-xs font-semibold hover:bg-blue-700 transition-all flex items-center gap-1"
                      >
                        <Download size={14} /> DL
                      </button>
                    )}
                    
                    <button
                      onClick={(e) => {
                        e.stopPropagation();
                        deleteFile(file.id);
                      }}
                      className="text-red-500 hover:bg-red-50 p-1 rounded-lg transition-all ml-auto"
                    >
                      <Trash2 size={16} />
                    </button>
                  </div>
                </div>
              ))
            )}
          </section>
        </div>

        {/* 右側：プレビュー */}
        <div className="col-span-3">
          <section className="bg-white p-6 rounded-2xl shadow-lg">
            <h2 className="text-lg font-semibold mb-4 flex items-center gap-2 text-gray-800">
              <Eye size={22} className="text-blue-600" /> プロトタイププレビュー
            </h2>
            
            {selectedFile ? (
              <div className="space-y-4">
                <div className="flex items-center justify-between pb-3 border-b">
                  <div>
                    <p className="font-bold text-gray-800">{selectedFile.name}</p>
                    <p className="text-xs text-gray-500">
                      {selectedFile.size} KB ? {selectedFile.uploadDate}

                  </div>

                    <iframe 
                      srcDoc={selectedFile.prototypeHtml}

                    <p className="font-semibold text-gray-700 mb-2">プロトタイプ未生成</p>
                    <p className="text-sm text-gray-500 mb-4">「変換」ボタンをクリックしてプロトタイプを生成してください</p>
                    <button
                      onClick={() => convertToPrototype(selectedFile.id)}
                      disabled={isProcessing}
                      className="bg-purple-600 text-white px-6 py-2 rounded-lg font-semibold hover:bg-purple-700 transition-all flex items-center gap-2 mx-auto disabled:opacity-50"
                    >
                      <Wand2 size={18} /> 
                      {processingId === selectedFile.id ? 'AI処理中...' : 'プロトタイプを生成'}
                    </button>
                  </div>
                ) : (
                  <div className="bg-gradient-to-br from-gray-50 to-gray-100 rounded-xl p-12 text-center border-2 border-dashed border-gray-300">
                    <div className="animate-spin rounded-full h-12 w-12 border-b-2 border-purple-600 mx-auto mb-4"></div>
                    <p className="font-semibold text-gray-700">AI処理中...</p>
                    <p className="text-sm text-gray-500 mt-2">PDFを解析してプロトタイプを生成しています</p>
                  </div>
                )}
              </div>
            ) : (
              <div className="text-center py-20 text-gray-400">
                <Eye size={48} className="mx-auto mb-3 opacity-30" />
                <p className="text-sm">ファイルを選択してください</p>
              </div>
            )}
          </section>
        </div>
      </div>
    </div>
  );
};

export default PDF2Prototype;

const axios = require('axios');
const cheerio = require('cheerio');

async function fetchAndParseNews(url) {
    try {
        // 1. 指定されたURLからページの全データを取得する
        // axios.get(url) は、サーバーに「中身をください」と非同期でリクエストを送る役割
        const response = await axios.get(url);
        const html = response.data;

        // 2. 取得したHTMLデータを、Cheerio（jQueryライクな操作ツール）でロードする
        // cheerio.load は、HTMLの山から特定の要素をセレクタで探せるようにする準備
        const $ = cheerio.load(html);

        const results = [];

        // 3. ニュースの見出しが格納されている「article」タグをすべて探し出す
        // $('article').each は、見つかったすべてのarticleに対し繰り返し処理を行う
        $('article').each((index, element) => {
            // 4. articleタグの中から、「h2」タグ（見出し）のテキストを取る
            // .text() は、タグを除去し中身だけを抽出し、余計な空白を .trim() で消す
            const title = $(element).find('h2').text().trim() || "No Title";

            // 5. aタグ（リンク）を探し、その中の 'href' 属性（URL）を抜き出す
            // .attr('href') は、<a href="..."> の属性値を特定して取得する
            const link = $(element).find('a').attr('href') || "";

            // 6. 抽出したデータをオブジェクトにして配列に追加する
            results.push({
                title: title,
                link: link
            });
        });

        return results;

    } catch (error) {
        // ネットワークエラーや404エラーなどが発生した場合の処理
        console.error("エラーが発生しました:", error.message);
        return [];
    }
}

def add_summary_row(excel_file):
  # 1. 読み込み
  df = pd.read_excel(excel_file)
  
  # 2. 最終行を計算
  last_row = len(df) + 1

  # len(df) は現在の行数。これが新しい末尾のインデックスになる

  # 4. 保存
  df.to_excel(excel_file, index=False)

  """ツールを実行し、成功なら結果を、失敗ならエラー内容を返す"""
  try:
    if tool_name == "read_file":
      # 正常な処理をここで呼び出す

    elif tool_name == "calculation":
      # 計算処理など

  except Exception as e:

    return f"【実行エラー】種類: {type(e).__name__}, 内容: {str(e)}"

import os

def perform_read_file(path):
  # ファイルが存在するかチェック
  if not os.path.exists(path):
    # フォルダ内のファイル一覧を取得してエラーに含める
    existing_files = os.listdir('.')
    raise FileNotFoundError(f"ファイル '{path}' が見つかりません。現在のディレクトリには {existing_files} があります。")
    
  # 正常な読み取り処理
  with open(path, 'r', encoding='utf-8') as f:
    return f.read()

def read_file(path):
  """ファイルの内容を読み取って返す"""

  return content

def replace_line(path, old_text, new_text):
  """ファイル内の特定の文字列を置換する"""
  print(f"  [System] ファイル '{path}' の内容を書き換えています...")
  content = read_file(path)

  if old_text not in content:
    return f"エラー: '{old_text}' がファイル内に見つかりません。"
      
  new_content = content.replace(old_text, new_text)
  with open(path, 'w', encoding='utf-8') as f:
    f.write(new_content)
  return f"成功: '{old_text}' を '{new_text}' に書き換えました。"

import fs from 'fs/promises';

/**
 * ログファイルを読み取り、エラー行を抽出して別ファイルに保存するツール
 */
async function analyzeLogTool(inputPath, outputPath) {
  try {
    // 1. ファイルを非同期で読み込む
    // fs.readFile(パス, エンコード) : ファイルの中身を文字列として取得
    const data = await fs.readFile(inputPath, 'utf-8');

    // 2. 内容を1行ずつ分割してエラーを探す
    // data.split('\n') : 改行コードで区切って配列にする
    // filter() : 条件に合う行（ここでは "ERROR" を含む行）だけを残す
    const errorLines = data.split('\n').filter(line => line.includes('ERROR'));

    // 3. 抽出結果を新しいファイルに書き出す
    // join('\n') : 配列を再び改行でつないで一つの文字列にする
    const resultText = `--- エラーレポート ---\n${errorLines.join('\n')}`;
    await fs.writeFile(outputPath, resultText);

    return `成功：${errorLines.length}件のエラーを抽出して ${outputPath} に保存しました。`;
  } catch (error) {
    // エラーハンドリング：ファイルがない場合などの例外をキャッチ
    return `失敗：ログの解析中にエラーが発生しました（${error.message}）`;
  }
}

import fs from 'fs/promises';

/**
 * 指定されたディレクトリ内のファイル一覧とサイズを取得するツール
 */
async function listFilesTool(dirPath) {
  try {
    // 1. ディレクトリ内のファイル名一覧を取得
    // fs.readdir(パス) : フォルダの中にあるファイル名の配列を返す
    const files = await fs.readdir(dirPath);

    // 2. 各ファイルの詳細情報（サイズなど）を取得
    // Promise.all : 複数の非同期処理（ファイルのステータス確認）を並列で実行
    const fileDetails = await Promise.all(files.map(async (file) => {
      const stats = await fs.stat(`${dirPath}/${file}`);
      return `${file} (${stats.size} bytes)`;
    }));

    return `ディレクトリ ${dirPath} の内容：\n${fileDetails.join('\n')}`;
  } catch (error) {
    return `エラー：ディレクトリを読み取れませんでした。パスを確認してください。`;
  }
}

from io import StringIO
from mcp.server.fastmcp import FastMCP

mcp = FastMCP("AdvancedAnalyzer")

@mcp.tool()
def aggregate_csv_data(csv_content: str, group_by_column: str, agg_column: str, method: str = "sum") -> str:
  """
  CSVデータを受け取り、指定された列でグループ化し、集計を行います。
  
  Args:
    csv_content: CSV形式のテキストデータ。
    group_by_column: グループ化の基準となる列名。
    agg_column: 集計対象となる数値列名。
    method: 集計方法（'sum', 'mean', 'count'）。
  """

    # 1. 文字列をDataFrameに変換
    df = pd.read_csv(StringIO(csv_content))
    
    # 2. 型のチェック
    if agg_column not in df.columns or group_by_column not in df.columns:
      return f"エラー: 指定された列名が見つかりません。存在する列: {list(df.columns)}"
        
    # 3. 動的な集計処理
    if method == "sum":
      result = df.groupby(group_by_column)[agg_column].sum()
    elif method == "mean":
      result = df.groupby(group_by_column)[agg_column].mean()
    else:
      result = df.groupby(group_by_column)[agg_column].count()
        
    return f"--- 集計結果 ({method}) ---\n{result.to_string()}"
      
  except Exception as e:
    return f"データ処理中にエラーが発生しました: {str(e)}"

if __name__ == "__main__":
  # サーバーを起動
  mcp.run()

# 操作を許可するディレクトリを限定（ホワイトリスト）

SAFE_DIRECTORY.mkdir(exist_ok=True)

@mcp.tool()
def safe_read_data(filename: str) -> str:
    """user_dataフォルダ内のファイルを安全に読み取ります。"""
    
    # 1. パスの正規化と検証

    try:
        return target_path.read_text(encoding='utf-8')
    except Exception as e:
        return f"エラー: ファイルの読み込みに失敗しました: {str(e)}"

import asyncio
from concurrent.futures import ProcessPoolExecutor
from mcp.server.fastmcp import FastMCP

mcp = FastMCP("PerformanceServer")
executor = ProcessPoolExecutor(max_workers=2)

def heavy_computation(data):
    # ここに時間のかかる複雑な計算を書く
    import time
    time.sleep(5) 
    return sum(data)

@mcp.tool()
async def run_heavy_task(data_list: list[float]) -> str:
    """重い計算をバックグラウンドで実行し、エージェントの停止を防ぎます。"""
    loop = asyncio.get_running_loop()
    
    try:
        # 1. タイムアウトを設定して別のプロセスで実行
        result = await asyncio.wait_for(
            loop.run_in_executor(executor, heavy_computation, data_list),
            timeout=10.0
        )
        return f"計算完了: {result}"
    except asyncio.TimeoutError:
        return "エラー: 計算時間が長すぎるためタイムアウトしました。"

import { McpServer } from "@modelcontextprotocol/sdk/server/mcp.js";

const server = new McpServer({
  name: "my-custom-server",
  version: "1.0.0",
});

// ツールを登録する
server.registerTool(
  "calculate_tax",
  {
    title: "税計算ツール",
    description: "金額と税率を受け取り、税込価格を計算します。",
    inputSchema: {
      amount: z.number(),
      taxRate: z.number(),
    },
  },
  async ({ amount, taxRate }) => {

    return {
      content: [
        {
          type: "text",
          text: `計算結果: 税込 ${total} 円です。`,
        },
      ],
    };
  }
);

async function main() {
  // stdio トランスポートを作成
  const transport = new StdioServerTransport();
  await server.connect(transport);
  console.error("Custom MCP Server running on stdio");
}

main().catch(console.error);
